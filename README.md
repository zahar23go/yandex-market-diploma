Yandex Market Diploma Project
📖 Описание
Yandex Market Diploma Project — дипломный проект по автоматизации тестирования интернет-магазина Яндекс Маркет. Проект демонстрирует навыки автоматизированного тестирования веб-приложений с использованием современных инструментов и подходов.

🎯 Цель проекта
Разработать комплексное решение для автоматизации тестирования ключевых функций интернет-магазина, включающее:

UI тестирование пользовательских сценариев (поиск, фильтрация, навигация, сортировка, корзина)

API тестирование бэкенд-сервисов (товары, категории, отзывы, поиск)

Автоматическую генерацию детальных Allure отчетов

Поддержку различных режимов запуска (parallel, headless, smoke, regression)
